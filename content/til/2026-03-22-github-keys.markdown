I've picked this up from a tutorial on using `age` - encryption tool.

A particular example there, in the docs is:

```
curl https://github.com/zygm0nt.keys | age -R ...
```

And I find this way of sharing keys convenient. 
Of course - this ties you to GitHub, which you might want to avoid
nowadays, but I believe you're using the service for some things.
